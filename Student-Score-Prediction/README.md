# Student Score Prediction

This is a Machine Learning project built while learning **Linear Regression**.

The model predicts a student's math score using:

* Gender
* Race/Ethnicity
* Test Preparation Course
* Reading Score
* Writing Score

## What I Learned

* Data preprocessing
* One-Hot Encoding
* Feature Scaling
* Linear Regression
* Scikit-learn Pipelines
* Model Serialization using Pickle

## Performance

* R² Score: 0.88

## Technologies

* Python
* Pandas
* Scikit-learn
